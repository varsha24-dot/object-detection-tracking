# Object Detection and Tracking

## Description
This project performs real-time object detection and tracking using YOLOv8, OpenCV, and Deep SORT.

## Features
- Real-time webcam/video detection
- Bounding boxes around objects
- Object tracking with unique IDs
- Real-time display output

## Technologies Used
- Python
- OpenCV
- YOLOv8
- Deep SORT
